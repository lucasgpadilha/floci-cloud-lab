# Observability

## Goal

Show how serverless systems are monitored using AWS-shaped APIs while running locally through Floci.

## Planned signals

- Application health checks.
- Lambda logs where implemented.
- API error smoke tests.
- DynamoDB/S3 operation outputs.
- CloudWatch Logs/Metrics interactions where supported.

## Evidence

Sanitized outputs may be saved under `evidence/` to show reproducible validation for the portfolio.
